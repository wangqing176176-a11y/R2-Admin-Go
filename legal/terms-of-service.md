# R2 Admin Go 用户协议

更新日期：2026-02-15

一、协议范围与接受

（一）本协议适用于你访问和使用 R2 Admin Go（以下简称“本平台”）的全部行为。  
（二）你注册、登录、访问或继续使用本平台，即表示你已阅读、理解并同意本协议及相关规则。  
（三）若你不同意本协议任一条款，请立即停止使用本平台。

二、平台服务说明

（一）本平台部署于 Cloudflare Pages，不提供自建服务器托管。  
（二）本平台主要功能为：帮助用户管理其自有 Cloudflare R2 存储桶中的对象文件。  
（三）本平台当前涉及的第三方服务包括但不限于：  
1. Cloudflare Pages：提供网站托管与网络分发；  
2. Cloudflare R2：存储桶与对象存储服务；  
3. Resend：邮箱验证码发送服务；  
4. Supabase：账号体系（注册/登录）及平台业务数据存储。

三、账号与安全

（一）你应保证注册信息真实、准确、完整，并及时更新。  
（二）你应妥善保管账号、密码、验证码及访问凭证，因你保管不当导致的风险和损失由你自行承担。  
（三）你不得将账号出借、出租、出售或以其他方式提供给第三方使用。  
（四）你发现账号存在异常登录或安全风险时，应立即联系平台运营者处理。

四、用户数据与授权

（一）你确认对所管理的 R2 桶及对象数据拥有合法权利，并已获得必要授权。  
（二）你理解并同意：为实现功能，本平台可能处理你填写的桶绑定信息及 R2 访问凭证信息。  
（三）你应自行确保所配置的 Access Key/Secret Key 权限范围最小化，避免使用高风险或超范围权限。  
（四）你应自行对重要数据进行备份，本平台不对你未备份导致的数据损失承担责任。
